# Aura ML

Aura ML is a real-time market intelligence and paper trading platform.

## Features
- Live market data (yfinance → Kite later)
- ML-driven insights
- Manual & algo paper trading
- Real-time analytics & BI
- REST APIs for developers

## Structure
- backend/  → APIs, ML, trading, analytics
- frontend/ → Binance-style UI
- docs/     → architecture & specs
